## Publish Commands

### Option 1: Local Publish (if you have npm credentials)
```bash
cd ~/GitHub/rosetta
npm login  # Enter your npm credentials if not already logged in
npm publish --access public
```

### Option 2: Trigger GitHub Actions CI Publish
For the CI-based approach, the release.yml workflow can run the publish step automatically:
```bash
cd ~/GitHub/rosetta
# Requires: a valid Confluence HTML export URL and CI NPM_TOKEN secret configured
gh workflow run release.yml \
  -f html_url="https://box.example.com/documents-export-2026-03-25.zip&dl=1" \
  -f version="v0.6.2" \
  -f force=true
```
